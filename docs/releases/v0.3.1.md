# SiteFueler v0.3.1 — Shell & Button Polish

**Type:** Patch (visual/UX polish on the shell + buttons)
**Tag:** `v0.3.1`

A polish pass after the v0.3.0 application shell. No new milestones — refinements
to typography, header, footer, navigation, and buttons.

## Highlights

### Typography
- **Instrument Sans** site-wide (Bunny Fonts, no JS); weight tokens clamped to 400–700.

### Header
- Clean, slim header (navbar **80 → 64px**), **SVG wordmark logo**, left-aligned
  light-weight nav, **no search**, cart icon (stroked), Login + Get Started.
- **Dropdown submenus** (desktop) for Templates / Plugins / Services.
- **Right-side off-canvas drawer** (mobile) with **accordion submenus**, a fuller
  menu set, CTAs, and a **social follow** row; smoother slide easing.

### Footer
- Overlapping brand-gradient **CTA card** (quote form on the Form System) + dark
  three-column footer (Best Selling / Useful Links) + bottom bar; white logo.

### Buttons
- **Top-only inset shadow** on all buttons (no bottom, no outer shadow).
- Filled variants (Primary/Danger) get a **darker border** (hover shade) so the
  edge reads clearly on light backgrounds.
- Filled buttons keep **white text on hover** (fixed the orange Get Started hover).
